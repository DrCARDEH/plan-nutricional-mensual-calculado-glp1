# plan-nutricional-mensual-calculado-glp1
Plan GLP-1 es un repositorio clínico y tecnológico diseñado para la gestión integral, titulación de dosis y seguimiento de pacientes en tratamiento metabólico y de obesidad con agonistas de GLP-1. Centraliza algoritmos de dosificación, control antropométrico y monitoreo de adherencia para optimizar la práctica médica y los resultados.
